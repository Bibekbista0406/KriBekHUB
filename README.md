## 🚀 Installation

```bash
git clone https://github.com/Bibekbista0406/KriBekHUB.git
cd KriBekHUB
chmod +x kri.sh clean.sh
sudo apt update && sudo apt install -y php curl wget unzip ssh
```
🖥️ Usage
```bash
# Start tool
bash kri.sh
```
```bash
# Clean traces
bash clean.sh
```
