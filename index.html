Installation

Get started with KriBekHUB in just a few simple steps


# Clone the repository
git clone https://github.com/Bibekbista0406/KriBekHUB

# Navigate to the directory
cd KriBekHUB

# Run the main script
bash kri.sh

Cleanup

When you're done testing, clean up all traces with:

bash clean.sh
